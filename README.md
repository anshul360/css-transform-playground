# CSS Transform Playground

App to play around with different CSS tranforms and the imapct of each using transform matrix.

## Getting Started

First, install all deps:

```bash
npm install
# or
yarn
# or
pnpm
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

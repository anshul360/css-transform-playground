# TransformLab

A visual sandbox for mastering CSS transforms, matrices, and anchor dynamics.

## Getting Started

First, install all deps:

```bash
npm install
# or
yarn
# or
pnpm
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
